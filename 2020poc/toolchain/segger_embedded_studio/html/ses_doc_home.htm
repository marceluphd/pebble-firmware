SEGGER Embedded Studio Reference Manual

This is the online documentation set for SEGGER Embedded Studio version 4.52.

Navigating the Documentation

You can use the contents explorer to browse the topics and pages contained within the documentation.

To select a particular page simply click on the page title.

You can resize, hide and show the contents explorer using the splitter bar.


Move backwards and forwards through the documentation one page at a time using the navigation links.


Use the breadcrumb trail to quickly find the current page’s position in the documentation hierarchy.


Search for keywords in the documentation using the search facility.
